SSH Library for Robot Framework
===============================

SSH Library is a test library for Robot Framework that enables execution of 
commands through SSH connection. It also enables file transfer to and from a
remote machine using SFTP protocol. 

For instructions about using test libraries with Robot Framework, see[1].
Installation instructions are available at [2].

[1] http://robotframework.googlecode.com/svn/trunk/doc/userguide/RobotFrameworkUserGuide.html#using-test-libraries
[2] http://code.google.com/p/robotframework-sshlibrary/wiki/InstallationInstructions
